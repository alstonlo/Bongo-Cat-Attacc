This file details expected protocol communication between the client and server


To register a new account:
Client sends:
    methodcallprotocol
    METHOD_FIELD: "register"
    "id": <give an id>
    "username": <username of account>
    "password": <password of account>

if successful, server sends
    returnprotocol
    RESPONSE_FIELD: <the id given above>
    "successful": "true"

if unsuccessful, server sends
    exceptionprotocol
